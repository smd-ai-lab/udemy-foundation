4. 🔌 MCP (Model Context Protocol)

Note : MCP est une spécification proposée par Anthropic pour découpler le contexte du prompt. En 2025, c’est devenu un standard de fait.

  • MCP Servers : exposent du contexte (fichiers, DB, API, métriques)

  • MCP Clients : agents/langchain qui lisent depuis un MCP server

  • Providers :

    • Serveur MCP pour Github, Slack, Notion, Gmail, Postgres

    • Frameworks : MCP SDK (Python/TS), mcp-agent

  • Outils compatibles MCP : Claude Desktop, Continue.dev, LangGraph MCP adapter

→ Remplacerait partiellement les « system prompts » par des contextes dynamiques et sécurisés
